# 🚀 Quick Start - Camera Detection

## 3 Simple Steps to Detect Animals!

### 1️⃣ Start Your Servers

```bash
# Terminal 1 - Backend
cd backend
node server.js

# Terminal 2 - Frontend  
cd frontend
npm run dev
```

### 2️⃣ Open Camera Page

**Manager:**
```
http://localhost:5173/dashboard/camera
```

**Admin:**
```
http://localhost:5173/admin/camera
```

### 3️⃣ Start Detection

1. Click **"Start Camera"** → Allow permissions
2. Click **"Start Detection"** → AI starts analyzing
3. Show elephant/tiger image to camera
4. **Detection happens automatically!** ✨

---

## 🎯 What You'll See:

- **Label on video** - "Elephant (85.2%)"
- **Stats update** - Counters increment
- **Recent detections** - Right panel updates
- **Console log** - "Posted detection"
- **Email sent** - Notification delivered
