## 2. INFORMATION WE COLLECT

### 2.1 Personal Information
We collect the following personal information when you register:

**Required Information:**
- **Full Name** - To identify you in the system
- **Email Address** - For account verification and alerts
- **Password** - Encrypted for account security
- **Plantation/Property Name** - To create and manage your property

**Optional Information:**
- **Phone Number** - For SMS/WhatsApp alerts (E.164 format)
- **Property Location** - For detection context
- **Property Address** - For property management

### 2.2 Automatically Collected Information

**User ID:**
- Each user is assigned a unique User ID (e.g., 001, 002, 003)
- This ID is associated with all your detections
- User ID is displayed in reports and notifications

**Detection Data:**
- **Animal Type** - Species detected (e.g., tiger, elephant, camel)
- **Confidence Score** - AI model confidence percentage
- **Detection Timestamp** - Date and time of detection
- **Property Name** - Associated property
- **Location** - Detection location (if available)
- **Source** - Detection method (live camera or uploaded image)
- **Detection Images** - Captured frames for verification

**System Data:**
- **IP Address** - For security and fraud prevention
- **Browser Type** - For compatibility
- **Device Information** - For optimization
- **Login Timestamps** - For security monitoring
- **Session Data** - JWT tokens for authentication

### 2.3 Camera and Media
- **Camera Access** - Required for real-time detection
- **Webcam Feeds** - Processed in real-time, not stored permanently
- **Uploaded Images** - Stored for detection and reporting
- **Detection Screenshots** - Saved for audit and reports

### 2.4 Communication Preferences
- **Alert Preferences** - SMS, WhatsApp, Voice call settings
- **Quiet Hours** - Your preferred do-not-disturb times
- **Critical Only Mode** - Alert filtering preferences
- **Email Preferences** - Notification settings

## 4. HOW WE SHARE YOUR INFORMATION

### 4.1 Within SADS System

**Admin Users:**
- Can view all user information for system management
- Can access all detection data across properties
- Can send notifications to managers
- Can manage user accounts and permissions

**Manager Users:**
- Can only view their own profile and detection data
- Can access their assigned property information
- Cannot view other managers' data

### 4.2 Third-Party Service Providers

**Twilio (SMS/WhatsApp/Voice):**
- Phone numbers for message delivery
- Message content (detection alerts)
- Delivery status and logs
- **Purpose:** Send SMS, WhatsApp, and voice call alerts
- **Privacy Policy:** https://www.twilio.com/legal/privacy

**Gmail SMTP (Email):**
- Email addresses for message delivery
- Email content (welcome messages, alerts, notifications)
- **Purpose:** Send email notifications
- **Privacy Policy:** https://policies.google.com/privacy

**MongoDB (Database):**
- All user data and detection records
- Encrypted passwords (bcrypt)
- **Purpose:** Data storage and management
- **Privacy Policy:** https://www.mongodb.com/legal/privacy-policy

### 4.3 We Do NOT Share
- ❌ We DO NOT sell your personal information
- ❌ We DO NOT share data with advertisers
- ❌ We DO NOT use your data for marketing to third parties
- ❌ We DO NOT share camera feeds with unauthorized parties

### 4.4 Legal Requirements
We may disclose your information if required by:
- Court orders or legal processes

## 5. DATA SECURITY

### 5.1 Security Measures
We implement industry-standard security measures:

**Encryption:**
- **Passwords** - Hashed using bcrypt (10 rounds)
- **Tokens** - JWT (JSON Web Tokens) for authentication
- **HTTPS** - Secure communication (in production)
- **Database** - MongoDB with authentication

**Access Controls:**
- **Role-Based Access** - Admin and Manager permissions
- **Session Management** - Automatic logout after inactivity
- **Rate Limiting** - Protection against brute force attacks
- **CORS** - Cross-origin request restrictions

**Monitoring:**
- **Error Logging** - System error tracking
- **Audit Trails** - User action logging
- **Security Alerts** - Suspicious activity detection

### 5.2 Data Retention
- **User Accounts** - Retained until account deletion
- **Detection Data** - Retained for reporting and compliance
- **System Logs** - Retained for 90 days
- **Backup Data** - Retained for disaster recovery

### 5.3 Your Responsibilities
- Keep your password confidential
- Use a strong, unique password
- Log out from shared devices
- Report suspicious activity immediately
- Keep your contact information updated
