## 📊 Accuracy Deep Dive

### Browser Detection (MobileNet):
```
Elephant Detection:
├─ Searches for word "elephant" in predictions
├─ May match: "African elephant", "Indian elephant"
├─ Accuracy: 75% (good lighting), 60% (poor lighting)
├─ False positives: Sometimes confuses large animals
└─ Best for: Quick alerts

Tiger Detection:
├─ Searches for "tiger", "cat", "leopard"
├─ May match: "tiger", "tiger cat", "leopard"
├─ Accuracy: 70% (clear view), 55% (partial view)
├─ False positives: Sometimes house cats
└─ Best for: General awareness
```

### YOLO Detection (YOLOv8):
```
Elephant Detection (Class 22):
├─ Specifically trained on elephants
├─ Recognizes shape, texture, size
├─ Accuracy: 95% (good lighting), 90% (poor lighting)
├─ False positives: Rare (<5%)
└─ Best for: Verification

Tiger/Big Cat Detection (Class 23):
├─ Trained on cats (includes big cats)
├─ Distinguishes by size, pattern
├─ Accuracy: 90% (clear view), 85% (partial view)
├─ False positives: Very rare (<3%)
└─ Best for: Official records
```
